# 一个简单的web文件服务器。
## 功能定位
* 可以用来在终端之间（pc，phone等等）互传数据。把想共享的文件（文件夹）放在file目录下，即可实现局域网内的共享。
## 使用说明
* file目录不可删除，为根目录。
* file目录下的favicon.icon为网站图标。可以删除或替换。
* url将被映射到file下。
* window系统双击start.bat启动服务
* 在浏览器中输入本机IP + ":8080"（例如192.169.32.43:8080)）即可访问。
* 若是在本机运行，则可以写localhost:8080
